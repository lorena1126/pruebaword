If you are seeking a theme that will immerse you to the world of make-believe, we might have exactly what you've been looking for. Thus, if you are a creative thinker, a photographer, for instance, who prefers to examine the world through camera lens, or a writer, who wishes their website to be an extension of their stories, Central is definitely a theme for you. To see how your site will look with Central, visit http://wp-demo-theme.bestweblayout.com/central/

== Installation ==

Manual installation:

1. Upload the `central` folder to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Themes' menu in WordPress.

Installation using "Add New Theme":

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for 'Central'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button
5. Activate the theme through the 'Themes' menu in WordPress

== Theme Notes ==

= SLider =

To have the slider shown on your website, you need to follow these 3 steps:
1. First, create a new category or choose one of the existing categories of posts, which you want to be displayed in the slider section on your home page.
2. If a selected category does not have posts yet, you need to create several posts. In the slider, post title, content and featured image of the post will be displayed ).
   NB: To be displayed in the slider, your posts must have a "draft" status. Therefore, you need set this status for every post  you want to see in slider. 
3. Navigate to "Apperance"->"Slider" menu item in Admin UI and choose a category of posts which you want to be displayed in the  slider.
( for featured images of the slider, we recommend you to use the proportions 900x218px (width/height) ).

= Featured Images =

For featured images of the posts or pages, we suggest you use the proportions 620x195px (width/height).

= Widgets =

The Theme includes standard widgets.
The following widgets are displayed in the sidebar by default: search form, recent posts, recent comments, archive and categories.
You are free to change the set of widgets as you like.
